Making your own Express server

When starting an Express server, install the express library. (npm i express or..)
Create package.json and insert {} into the file to make it a real json file.
insert the following before running npm install

{
"dependencies": {
  "express": "4.17.2"
   }
}

After that, import Express into your app inside of app.js

const express = require('express')
const app = express();

and insert the following in the bottom of app.js to make the server listen on a given port:

app.listen(8080,(error) => {
console.log("Server is running on port", 8080);
}); //add a callback if wanted.

This is a basic express setup.
Since this is a basic backend, we will be creating the following http requests:
GET, GET(id), POST, PUT, PATCH, DELETE

Reading JSON

We can predict that we are going to be sending JSON bodies, so we are going to need express.json()
express.json() allows for body parsing into json.
If we forgot this step, our request.bodies would be undefined(null) because it can't understand json yet.
This is because the data has been sent in a bitestream and hasn't been told how to be rewritten.
We could also get around this issue by using http headers, such as CONTENT-TYPE: 'application/json'
