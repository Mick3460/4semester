TYPE COERCION:

When checking if a given variable is the same, we are used to using the logical operator == from java to check, if they are "the same".
However, in javascript, we don't have a compiler to tell us if two given variables are the same type when writing the code.
Therefore, if we want to make a hard check, to see if 2 and '2' are the same, we would be using ===
This makes sure that the two "numbers" are actually both numbers, and not just a number and a parsed string.
The strict equality === doesn't implicitly convert any of the values.

STRINGS AND NUMBERS:

Parsing strings with numbers into floats or Numbers: Whenever a number is returned as a string, convert it using Number(string) or parseFloat(string). Using

Number()

instead of parse to make sure it's a "clean" number, since parseInt(123asd2) would return 123 only.

Print a number with decimals with

number.toFixed(amountOfDecimals)

When using a string, you can get a character by it's index with:

string.charAt(index)

If you want to replace a character or substring, you can use

string.replace("thingToReplace", "thingThatReplaces")

OBJECTS:

If you wanna access a value by a key, you can use:

object.keyname

When defining an object, use this syntax

myObject = { key: value, key: "value" }

You can further increase the objects keyvalue pairs with

myObject.newKey = newValue;

Or delete it with

delete myObject.key

ARRAYS:

You can access arrays directly with indexes:

array[indexToAccess]

If you need more objects in the array, add with with push

array.push( newObject . or maybe {object in here} )

Get the index of first occurrence of that value.

array.indexOf(thingToFind). Returns -1 if not found.

If you want to add to an array on a specific spot, use splice. Splice can also remove.

array.splice(startIndex to insert/delete, how many things to replace, "stringToInsert")

pop() can be used to remove the last in an array

array.pop()

If you want to copy an array, either slice it, map it or use the spread operator.

newArray = array.slice( (optional start), (optional end) )
newArray = array.map ( x => x)
newArray = [...array]

FUNCTIONS:

Anonymous functions are nameless functions, such as

const variable = function (x,y) { return x+y }
const variable = (x,y) => {return x+y}
const variable = (x,y) => x+y in oneliners

Functions are hoisted when a script is run. This makes it possible for javascript to read and call methods that are declared further down in the script.

When using functions, you can pass another function as an argument. This is called a callback function. such as:

function someAction(anotherFunction, variable) { anotherFunction(variable)}

LOOPS:

When using loops in javascript, avoid for loops as it's errorprone because of int indexing. Furthermore, they are overly verbose and aren't functional. Luckily there are loop methods that solve these problems We are looking for a way to have immutable data, so avoid operating directly inside arrays. If you need to iterate and "throw away" the data, you can use a forEach. If you need immutable data structures or copies of arrays, use map (or spread operator)

const discountedPhones = smartPhones.map(smartPhone => {
   return {...smartPhone, price: smartPhone.price - 500 }; });

You can also filter out elements you dont want. filter() returns a new array, same goes for map.

const cheapPhones = smartPhones.filter(phone => phone.price <= 4000)
