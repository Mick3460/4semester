Using res.send to send html

When using the send() method, it's possible to send back plain html and have it rendered by the browser.

app.get("/welcome", (req, res) => {
    res.send(`<h1>WELCOME ${req.params.name} </h1>
  `);
});

Enable the public folder

Building upon what we already know, we will now need a way for our server to be able to see local files within the app.
This can be done with:

app.use(express.static("public"))

Where we define the "static" folder to be the folder called public. This allows for usage of files inside of the folder.

Sending files instead of text

Now that we have certain files allowed, we can have templates and html files that can be sent back as a response.
We just have to alter the response.send() method to this.

res.sendFile(__dirname + "/public/frontpage/frontpage.html");

To make the path compatible on all systems, use the absolute path within the app.
The __dirname in a node script returns the path of the folder where the current JavaScript file resides.
So basicly, dirname is the place where the project is being run from, and it is a global variable within the application, thanks to javascript.
